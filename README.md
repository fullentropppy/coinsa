**Coinsa** - iOS-приложение для отслеживания расходов в путешествиях. Помогает планировать расходы, учитывать траты на локациях и получать понятную аналитику по бюджету.

## Основные возможности

### Управление поездками
- Создание поездок с выбором **основной валюты** (в ней хранятся все бюджеты и траты, вводятся и отображаются суммы)
- Добавление **локаций** внутри каждой поездки c выбором местной валюты (в ней, как и в основной валюте, вводятся и отображаются суммы)

### Бюджет и траты
- **Планирование бюджета** для каждой локации по категориям:
  - Еда
  - Транспорт
  - Активность
  - Шоппинг
  - Медицина
  - Другое
- **Учёт трат** по этим категориям на локациях, с указанием способа оплаты:
  - Наличные
  - Карта

### Валюты и курсы
- **Автоматическая загрузка актуальных курсов валют**
- Возможность задать **процент корректировки курса** (например, для учёта комиссии банка)

### Аналитика
- Простая и наглядная аналитика по каждой поездке и локации:
  - Запланированный бюджет
  - Фактические траты
  - Остатки по категориям

### Экран «Сегодня»
- Информация по текущему дню:
  - Рекомендованный бюджет на сегодня
  - Фактические траты за день
- Быстрый доступ к созданию новой траты

### Синхронизация
- **Автоматическая синхронизация данных через iCloud** — все данные приложения доступны на ваших устройствах Apple.
